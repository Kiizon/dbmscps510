# MOBA Game Database Dashboard

A web-based admin dashboard for managing a MOBA game database. Built with Flask and SQLite3, it provides a complete interface for managing players, matches, items, and game statistics across 12 interconnected database tables.

## How to Run

```bash
source env/bin/activate
python3 app.py
```

Then open http://127.0.0.1:5000/ in your browser.

## Quick Start

1. Click "Create Tables" to initialize the database
2. Click "Populate Sample Data" to add demo data
3. Use the sidebar to navigate between sections
